## Type

### FilterDataItem

| 参数     | 说明                  | 类型             | 可选值        | 默认值 |
| -------- | --------------------- | ---------------- | ------------- | ------ |
| title    | 选项的展示内容        | string           | -             | -      |
| id       | 选项对应的值，唯一 id | string           | -             | -      |
| disabled | 是否禁用              | boolean          | true \| false | false  |
| children | 级联筛选项列表        | FilterDataItem[] | -             | -      |
