# Result 结果页

用于反馈一系列操作任务的处理结果。

## 何时使用

- 当有重要操作需告知用户处理结果，且反馈内容较为复杂时使用。

## Props

| 参数             | 说明         | 类型            | 可选值        | 默认值 |
| ---------------- | ----------- | -------------- | ------------- | ------ |
| image            | 自定义图标    | ReactNode      | -             | -      |
| imageSize        | 图标尺寸      | string         | 'sm' \| 'md' \| 'lg' | 'md'  |
| type             | 组件类型      | string         | 'info' \| 'success' \| 'warning' \| 'error' | 'info'  |
| title            | 标题         | ReactNode      | -             | -      |
| content          | 内容         | ReactNode      | -             | -      |
