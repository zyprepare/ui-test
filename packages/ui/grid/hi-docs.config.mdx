## Type

### GridResponsiveSize<T>

| 参数     | 说明             | 类型      | 可选值        | 默认值 |
| -------- | ---------------- | --------- | ------------- | ------ |
| xs       | 设置屏幕 < 576px 时响应式栅格数 | T | -             | -      |
| sm       | 设置屏幕 >= 576px 时响应式栅格数 | T | -             | -      |
| md       | 设置屏幕 >= 768px 时响应式栅格数 | T | -             | -      |
| lg       | 设置屏幕 >= 992px 时响应式栅格数 | T | -             | -      |
| xl       | 设置屏幕 >= 1200px 时响应式栅格数 | T | -             | -      |
